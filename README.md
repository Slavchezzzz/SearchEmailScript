# YouTube Email Parser 🎯

Программа для автоматического поиска email-адресов в описаниях YouTube видео по заданному хештегу. Имеет удобный графический интерфейс, сохраняет историю проверенных видео и найденные email в Excel файл.

## 🚀 Возможности

- ✅ Поиск видео на YouTube по хештегу
- ✅ Автоматическое раскрытие описания видео
- ✅ Извлечение email-адресов из описания
- ✅ Сохранение найденных email в Excel файл
- ✅ Отслеживание проверенных видео (не повторяет уже обработанные)
- ✅ Удобный графический интерфейс
- ✅ Настройка количества обрабатываемых видео
- ✅ Логирование всех действий

## 💻 Требования

- **Операционная система**: Windows 10/11
- **Python**: версия 3.7 или выше
- **Браузер**: Google Chrome (последняя версия)
- **Интернет**: подключение для поиска видео

## 📦 Установка

### Шаг 1: Установка Python
1. Скачайте Python с [официального сайта](https://www.python.org/downloads/)
2. При установке **обязательно отметьте галочку** `"Add Python to PATH"`
3. Проверьте установку:
```bash
python --version
```

### Шаг 2: Скачивание программы
1. Создайте папку для проекта, например: `C:\YouTubeParser`
2. Скопируйте все файлы программы в эту папку

### Шаг 3: Установка библиотек
Откройте командную строку в папке с программой и выполните:

```bash
pip install -r requirements.txt
```

Или установите библиотеки по отдельности:
```bash
pip install PyQt5
pip install selenium
pip install pandas
pip install openpyxl
```

### Шаг 4: Установка ChromeDriver
1. Узнайте версию вашего Chrome:
   - Откройте Chrome
   - Нажмите ⋮ → Справка → О Google Chrome
   - Запомните версию (например: 120.0.6099.216)

2. Скачайте ChromeDriver:
   - Перейдите на [страницу загрузки](https://chromedriver.chromium.org/downloads)
   - Выберите версию, соответствующую вашему Chrome
   - Скачайте `chromedriver_win64.zip`

3. Установите драйвер:
   - Распакуйте архив
   - Скопируйте `chromedriver.exe` в папку `drivers` внутри проекта
   - Путь должен быть: `C:\YouTubeParser\drivers\chromedriver.exe`

## 📚 Библиотеки

### 1. **PyQt5** - для графического интерфейса
```bash
pip install PyQt5
```
**Назначение:** Создание окон, кнопок, таблиц, прогресс-баров и всех визуальных элементов.

**Компоненты:**
- `QApplication` - управление приложением
- `QMainWindow` - главное окно
- `QPushButton` - кнопки
- `QLineEdit` - поля ввода
- `QTableWidget` - таблица результатов
- `QProgressBar` - индикатор прогресса
- `QThread` - многопоточность

### 2. **Selenium** - для автоматизации браузера
```bash
pip install selenium
```
**Назначение:** Управление браузером Chrome, поиск элементов на странице, клики, ввод текста.

**Компоненты:**
- `webdriver` - управление браузером
- `WebDriverWait` - ожидание элементов
- `By` - поиск элементов (CSS, XPath, ID)

### 3. **pandas** - для работы с данными
```bash
pip install pandas
```
**Назначение:** Создание таблиц, обработка данных, работа с Excel.

**Компоненты:**
- `DataFrame` - таблица данных
- `read_excel()` - чтение Excel
- `to_excel()` - сохранение в Excel

### 4. **openpyxl** - для работы с Excel
```bash
pip install openpyxl
```
**Назначение:** Чтение и запись Excel файлов. Используется pandas для работы с .xlsx форматом.

## 🎮 Запуск программы

python main_gui.py

## 📖 Инструкция по использованию

### 1. **Запуск программы**
- Откройте программу, появится главное окно

### 2. **Ввод данных**
- В поле "Хештег" введите нужный тег (например: `#python`, `#music`, `#tutorial`)
- В поле "Макс. видео" выберите количество видео для обработки (1-20)

### 3. **Запуск поиска**
- Нажмите кнопку **"Старт"**
- Программа начнет открывать YouTube и искать видео

### 4. **Отслеживание прогресса**
- **Прогресс-бар** показывает текущий статус
- **Логи** отображают все действия программы
- **Таблица** заполняется найденными email

### 5. **Завершение работы**
- После обработки всех видео появится сообщение
- Результаты сохраняются автоматически

### 6. **Просмотр результатов**
- Найденные email: вкладка "Найденные email"
- История действий: вкладка "Логи"
- Файлы: папка `data` в директории программы
  - `youtube_emails.xlsx` - все найденные email
  - `checked_urls.json` - история проверенных видео

## 🔧 Устранение проблем

### Проблема 1: "Python не найден"
```bash
python не является внутренней или внешней командой
```
**Решение:** Добавьте Python в PATH или используйте:
```bash
py main_gui.py
```

### Проблема 2: "Нет модуля PyQt5"
```bash
ModuleNotFoundError: No module named 'PyQt5'
```
**Решение:** Установите библиотеку:
```bash
pip install PyQt5
```

### Проблема 3: "ChromeDriver не найден"
```bash
Message: 'chromedriver.exe' executable needs to be in PATH
```
**Решение:** 
1. Скачайте ChromeDriver с [официального сайта](https://chromedriver.chromium.org/)
2. Поместите `chromedriver.exe` в папку `drivers`

### Проблема 4: Программа закрывается сразу после запуска
**Решение:** Запустите через командную строку, чтобы увидеть ошибку:
```bash
python main_gui.py
```

## ❓ Часто задаваемые вопросы

### В: Где сохраняются найденные email?
**О:** В папке `data` создается файл `youtube_emails.xlsx`

### В: Программа будет проверять одни и те же видео повторно?
**О:** Нет! Все проверенные видео сохраняются в `checked_urls.json` и пропускаются при следующих запусках

### В: Сколько видео можно обработать за раз?
**О:** Можно выбрать от 1 до 20 видео

### В: Нужен ли интернет для работы?
**О:** Да, программа работает онлайн с YouTube

### В: Почему программа не находит email?
**О:** Возможно, в выбранных видео действительно нет email в описании
